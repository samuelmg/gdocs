# Gestor de Documentos

Control y seguimiento de documentos para dependencias.

# Instalación

1. ´git clone https://github.com/samuelmg/gdocs.git´
2. ´cd gdocs´
3. ´composer install´
4. ´cp .env.example .env´
5. ´php artisan key:generate´
6. `create DB`
7. `php artisan migrate`
